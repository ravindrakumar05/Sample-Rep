Java Programming Demo Project
Overview

This project is a simple demonstration of Java programming concepts. It shows how to write, compile, and run a basic Java application. The goal of this project is to help beginners understand the basic structure of a Java program.

Features

Simple Java program

Demonstrates basic syntax

Easy to compile and run

Beginner friendly

Technologies Used

Java

JDK 8 or above

Command Line / Terminal

Any IDE (Eclipse, IntelliJ IDEA, VS Code)
